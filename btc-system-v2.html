① 사이즈 계산 7단계

① 마진 의도 USDT  = S × M
   예: 200 × 50% = 100 USDT

② 노출 의도 USDT  = 마진 의도 × 3
   예: 100 × 3 = 300 USDT

③ BTC 수량 (이론) = 노출 의도 / P
   예: 300 / 78,400 = 0.003827 BTC

④ BTC 수량 (실제) = round_down(이론값, Lot Size)  ⭐ 내림 처리
   예: 0.003827 → 0.003 BTC (Lot 0.001 자동 내림)

⑤ 실제 노출 USDT   = BTC 수량(실제) × P
   예: 0.003 × 78,400 = 235.2 USDT

⑥ 실제 마진 USDT   = 실제 노출 / 3
   예: 235.2 / 3 = 78.4 USDT

⑦ 실제 시드 사용률 = 실제 마진 / S × 100
   예: 78.4 / 200 = 39.2%

② Stop 검증 4단계 (재기재)

① Stop 거리 %        = (P - SL) / P × 100
   예: (78,400 - 77,375) / 78,400 = 1.31%

② 예상 손실 USDT     = (P - SL) × BTC 수량(실제)
   예: (78,400 - 77,375) × 0.003 = 3.075 USDT

③ 예상 시드 손실률 = 예상 손실 / S × 100
   예: 3.075 / 200 = 1.54%

④ 검증:
