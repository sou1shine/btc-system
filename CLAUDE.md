### 매트릭스 (1~3점 / 합계 12점)
- 거래량 4H 5MA: 1.3~1.5x = 1점 / 1.5~2.0x = 2점 / 2.0x+ = 3점
- RSI 상태: > MA = 1점 / + > 50 = 2점 / + > 60 = 3점
- 캔들 강도: 음봉 고가 돌파 = 1점 / 평균 1.2x = 2점 / 1.5x or Engulfing = 3점
- 구조 정합성: 레인지 내부 = 1점 / 이평선 지지 = 2점 / 핵심 저항 돌파 = 3점

### 등급 → 사이즈
- S (11~12): 60~70% / Stop -3%
- **A (8~10): 45~55% / Stop -5%** ← 5차+ 주 사용
- B (6~7): 30~40% / Stop -7%
- 5점 이하: 진입 금지

### 3-Layer 청산
- Layer 1 (100% / Hard): 진입 근거 4H 종가 이탈 / Stop 발동
- Layer 2 (50% / Critical): 다음 3 중 2 충족 — (1) 4H 음봉 거래량 1.5x+ (2) 직전 4H 스윙 저점 이탈 (3) RSI < MA + RSI < 50
- Layer 3 (Warning): 일봉 EMA 20 이탈 / 4H 윗꼬리 30%+ / RSI 다이버전스
